How to publish this site

  1. Create a file named index.html in the repo root (or in /docs).
  2. Go to Settings → Pages.
  3. Set Source to the branch you want (usually main).
  4. Select the folder: /(root) or /docs.
  5. Save. GitHub will provide a public Pages URL.
